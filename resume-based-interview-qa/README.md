## 1. Django me ORM kya hota hai?

**English:**
ORM (Object-Relational Mapping) is a technique that lets us interact with the database using Python code instead of writing raw SQL queries. In Django, ORM allows us to create, retrieve, update, and delete
  database records using Python classes and methods. It makes database handling easier and safer.

**Hindi:**
ORM ek technique hai jisse hum database ko direct SQL likhne ki bajaye Python code se control karte hain. Django me ORM Python classes ke through tables banata hai aur data ko add, update, delete karna 
 simple bana deta hai. Ye development fast aur error-free hota hai.

## 2. FastAPI ka use kab aur kyun karte hain?

**English:**
FastAPI is used to build fast and modern web APIs with Python. It is chosen for its high performance, automatic data validation, and easy integration with Python type hints. FastAPI is great when you want 
 asynchronous programming and faster development.

**Hindi:**
FastAPI Python me fast aur modern web APIs banane ke liye use hota hai. Iska fayda ye hai ki ye high performance deta hai, data validation automatic karta hai aur Python type hints ko support karta hai. 
 Jab asynchronous aur fast development chahiye hota hai tab FastAPI best hota hai.

Ex:- Synchronous Programming kya hota hai?:-  Synchronous programming me kaam ek ke baad ek hota hai.Jab tak pehla kaam khatam nahi hota, doosra kaam start nahi hota.
Program wait karta hai.
Socho tum restaurant me khud jaake order dete ho:

Order diya -->Wahin khade rahe --> Khana mila tab aage badhe --> Ye synchronous hai — tum wait kar rahe ho.
Ex:-
🔹 Synchronous:
import time

print("Start")
time.sleep(5)   # waits 5 seconds
print("End")


Asynchronous Programming kya hota hai?:-  Asynchronous programming me task start hota hai lekin program wait nahi karta.
Jab tak task complete ho, tab tak doosra kaam karta rehta hai.
Ex:-
🔹 Asynchronous:
import asyncio

async def task():
    print("Start")
    await asyncio.sleep(5)
    print("End")

asyncio.run(task())
