1. Deep vs Shallow copy

👉 Shallow = reference copy 👉 Sirf outer object copy hota hai
👉 Andar ke elements ka reference same rehta hai
import copy 

a = [[1,2],[3,4]]
b = copy.copy(a) # Shallow copy

b[0][0] = 100

print(a)

👉 Output:- [[100, 2], [3, 4]]


👉 Deep = actual copy (independent)
Pure object + inner objects sab copy hote hai
👉 Completely independent

import copy

a = [[1,2],[3,4]]
b = copy.deepcopy(a)

b[0][0] = 100   # ✔ correct

print(a)

🎯 Output:- [[1, 2], [3, 4]]

### 12. Multiprocessing vs Multithreading
# multithreading and multiprocessing both are used to achive multitasking


👉 CPU tasks → multiprocessing
👉 I/O tasks jese(file,API,DB) → multithreading

🔹 Multithreading kya hota hai?
Multithreading matlab ek hi process ke andar multiple threads (chhote tasks) ek saath run hona.

Same memory share karte hain
Fast hota hai (lightweight)
Mostly use hota hai jab kaam I/O based ho (jaise file read, network call)

👉 Example:- Ek app jo ek hi time pe download bhi kar raha hai aur UI bhi smooth chala raha hai.

🔹 Multiprocessing kya hota hai?

Multiprocessing me ek time par multiple processes parallel run hote hai, isliye ye CPU-bound tasks ke liye best hota hai.

Har process ka apna memory hota hai
Heavy hota hai but powerful
Use hota hai jab kaam CPU intensive ho (jaise calculations, AI models)

👉 Example:
Video rendering, heavy data processing, gaming engines.

Multithreading = ek hi kitchen mein 5 log milke kaam kar rahe hain
Multiprocessing = 5 alag kitchens mein alag teams kaam kar rahi hain
